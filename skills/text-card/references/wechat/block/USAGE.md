# Block 色块公众号封面模板

色块拼贴风格公众号封面，左侧大标题 + 色块装饰，右侧圆角气泡面板 + 强调文字。视觉特点：几何色块、对话气泡面板、胶囊标签。

模板文件：`template.html`

## 卡片尺寸

- 总宽高：2680 × 800 px
- 宽高比：3.35:1
- 左侧区域（推送封面）：1880 × 800 px（2.35:1）
- 右侧区域（转发缩略图）：800 × 800 px（1:1）

> 公众号推送时，完整封面显示左侧 2.35:1 部分；用户转发到朋友圈/聊天时，显示右侧 1:1 方形缩略图。设计内容时需保证两部分各自独立可读。

## 渲染参数

```bash
--viewport-width 2680 --viewport-height 800
```

## 变量说明

### 颜色变量（CSS `:root`）

| 变量 | 说明 | 示例值（green） |
|------|------|-----------------|
| `--bg` | 卡片背景色（左右两侧共用） | `#f0f0ec` |
| `--block` | 左下角色块颜色 | `#2979ff` |
| `--panel` | 右侧圆角气泡面板背景色 | `#00d68f` |
| `--main-text` | 主文字颜色（headline / category / tag / accent-text / brand / bubble 背景） | `#1a1a1a` |
| `--bubble-text` | bubble 内文字颜色（需与 `--main-text` 形成反差） | `#ffffff` |

### 内容变量（HTML）

| 变量 | 位置 | 说明 |
|------|------|------|
| `{{category}}` | 左上角 | 分类描述文案 |
| `{{headline}}` | 左侧居中偏左 | 主标题，用 `<br />` 控制换行 |
| `{{tag}}` | 左侧右上方 | 胶囊标签（圆角边框） |
| `{{bubble}}` | 右侧面板顶部 | 气泡标签（深色底 + 下三角） |
| `{{accent_text}}` | 右侧面板居中 | 强调文字，用 `<br />` 控制换行 |
| `{{brand}}` | 右侧面板右下角 | 品牌/来源署名 |

### 装饰元素（不需要修改）

- **color-block**：左下角矩形色块（右上圆角），颜色由 `--block` 控制
- **panel-bubble**：右侧圆角面板，底部有三角形气泡尾巴，颜色由 `--panel` 控制
- **bubble 下三角**：bubble 标签底部的三角箭头，颜色跟随 `--main-text`

## 文案约束

| 变量 | 限制 | 原因 |
|------|------|------|
| `headline` | **建议 2 行，每行 ≤ 5 个中文字**；用 `<br />` 手动换行 | 字号 220px，左对齐但不自动换行，过长会与右侧区域重叠 |
| `accent_text` | **建议 2 行，每行 ≤ 2 个中文字**；用 `<br />` 手动换行 | 字号 180px，`white-space: nowrap`，右侧面板内空间有限 |
| `category` | **≤ 10 个中文字** | 字号 36px，位于左上角，过长可能与 tag 重叠 |
| `tag` | **≤ 6 个字符** | 胶囊标签，`white-space: nowrap`，内容过长会超出右侧边界 |
| `bubble` | **≤ 4 个中文字** | 字号 40px，`white-space: nowrap`，居中于面板顶部，过长会溢出面板 |
| `brand` | **≤ 8 个字符** | 字号 36px，位于面板右下角固定位置 |

### 特殊说明

- **`headline` 左对齐**：与 editorial / grid 的居中标题不同，headline 靠左排列，注意文案长度不要与右上角的 tag 重叠。
- **`accent_text` 建议极短文案**：右侧面板内同时有 bubble（顶部）和 brand（底部），中间留给 accent_text 的空间有限，2 行各 2 字是最佳比例。
- **`--block` 和 `--panel` 是两个独立的装饰色**：可以选择对比色或同色系，它们分别作用于左下角色块和右侧面板，是模板的主要视觉亮点。

## 示例样式

### green（灰白 × 蓝绿撞色）

文件：`example-green.html`

```
--bg: #f0f0ec          暖灰白背景
--block: #2979ff       明蓝色块
--panel: #00d68f       翠绿面板
--main-text: #1a1a1a   深黑文字
--bubble-text: #ffffff  白色气泡文字
```

## 配色技巧

- `--block` 和 `--panel` 选择两个高饱和的对比色（如蓝 + 绿、橙 + 紫），可以营造强烈的视觉冲击。
- `--bg` 建议使用低饱和的浅色（暖白、浅灰），让色块和面板成为视觉焦点。
- `--bubble-text` 需与 `--main-text` 形成反差（bubble 背景色就是 `--main-text`），通常深色文字配白色 bubble 文字。
