# Grid 网格公众号封面模板

网格背景 + 高亮色条 + 几何圆形装饰的公众号封面风格。适合公众号推送封面图、文章头图。视觉特点：细网格底纹、标题第二行自带有色高亮条背景、右上角双圆装饰。

模板文件：`template.html`

## 卡片尺寸

- 总宽高：2680 × 800 px
- 宽高比：3.35:1
- 左侧区域（推送封面）：1880 × 800 px（2.35:1）
- 右侧区域（转发缩略图）：800 × 800 px（1:1）

> 公众号推送时，完整封面显示左侧 2.35:1 部分；用户转发到朋友圈/聊天时，显示右侧 1:1 方形缩略图。设计内容时需保证两部分各自独立可读。

## 渲染参数

```bash
--viewport-width 2680 --viewport-height 800
```

## 变量说明

### 颜色变量（CSS `:root`）

| 变量 | 说明 | 示例值（orange） |
|------|------|------------------|
| `--bg` | 卡片背景色（左右两侧共用，叠加网格线） | `#f5f5f0` |
| `--highlight` | 高亮色（tagline 背景色条 + 右上角实心圆） | `#ec5a0c` |
| `--main-text` | 主文字颜色（headline / tagline / accent-text / 圆环描边） | `#1a1a1a` |
| `--label-text` | 辅助文字颜色（× 标记 / category 中的 small 副标签） | `#888888` |

### 内容变量（HTML）

| 变量 | 位置 | 说明 |
|------|------|------|
| `{{category}}` | 左上角 | 分类标签，支持用 `<small>` 嵌套副标签（副标签会自动变小号灰色） |
| `{{headline}}` | 左侧居中第一行 | 主标题，无高亮背景 |
| `{{tagline}}` | 左侧居中第二行 | 副标题，自带有色高亮条背景（颜色由 `--highlight` 控制） |
| `{{accent_text}}` | 右侧底部居中 | 强调文字 |

```html
<div class="category">工具评测 · CLI COMPARE</div>
```

### 装饰元素（不需要修改）

- **x-mark**：左上角 × 标记
- **circle-deco**：右上角双圆装饰（空心圆 + 实心填充圆，实心圆中央有 * 号）
- **网格背景**：左右两侧均有 52px 网格线叠加

## 文案约束

| 变量 | 限制 | 原因 |
|------|------|------|
| `headline` | **固定 1 行，≤ 4 个中文字** | 字号 210px，父容器 `white-space: nowrap` 不换行，过长会超出左侧区域 |
| `tagline` | **固定 1 行，≤ 4 个中文字** | 字号 210px，与 headline 相同约束；高亮条宽度跟随文字，字数过多会溢出 |
| `accent_text` | **≤ 4 个中文字** | 字号 150px，`white-space: nowrap` 不换行，右侧区域仅 800px 宽 |
| `category` | 主标签 **≤ 20 字符**，副标签 **≤ 10 字** | 固定在左上角，过长会与居中标题重叠 |

### 特殊说明

- **headline 和 tagline 字数应一致或接近**：两行上下居中对齐，字数差异过大会导致视觉不平衡。
- **深色背景时 tagline 文字颜色需注意**：模板默认 tagline 文字使用 `--main-text` 颜色。当背景为深色、`--highlight` 也为深色时，需要手动将 `.highlight-text` 的 `color` 和 `-webkit-text-stroke` 改为与 `--highlight` 对比的颜色（参考 teal 示例中的做法）。

## 示例样式

### orange（暖白 × 鲜橙）

文件：`example-orange.html`

```
--bg: #f5f5f0        暖白背景
--highlight: #ec5a0c  鲜橙高亮
--main-text: #1a1a1a  深黑文字
--label-text: #888888 灰色标签
```

## 配色技巧

- `--highlight` 是整个模板的视觉焦点色，同时作用于 tagline 色条和右上角实心圆，选择高饱和色效果最佳。
- 亮色背景（orange）：`--main-text` 用深色，tagline 文字默认就能在高亮条上保持可读。
- 深色背景时：`--main-text` 用浅色，但 tagline 文字需要手动改为深色（与 `--highlight` 形成反差），否则浅色文字在亮色高亮条上不清晰。
- 网格线透明度已内置（亮色背景用黑色 6% 透明度，深色背景用白色 4% 透明度），深色配色时可能需要调整网格线的 `rgba` 值。
